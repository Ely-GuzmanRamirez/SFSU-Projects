# Mushroom Classification Project

This project classifies mushrooms as edible or poisonous using machine learning models.

## Models Used

- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- Naive Bayes
- Quadratic Discriminant Analysis
- Bagging

## Main Result

Most models performed extremely well, with several reaching 100% test accuracy.
